This week introduces some common strategies for breaking a complex computational problem down into simpler sub-problems. These types of divide-and-conquer, recursive approaches are often useful.

Learning Objectives:
  • Explain common algorithm strategies
  • Use proof by induction to reason about programs
  • Implement recursive algorithms in code
