Logic gates are the foundation of the digital computer. This week you will learn how logic gates make up the processor and memory of a computer. This low-level understanding will help you reason about how your programs work.

Learning Objectives:
  • Distinguish the basic types of logic gate
  • Use logic gates to build simple circuits
  • Explain how computer memory works
